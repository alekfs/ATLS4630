# Pedal Stoke UX Audit Images

Please add the following images to this directory:

## Required Images

1. **hero.jpg** - The main hero image for the case study
   - Recommended: A screenshot of the Pedal Stoke homepage or the cover page from your audit PDF
   - Dimensions: Approximately 1200x700px
   - Format: JPG

## Optional Images

You can also add additional images from your audit report to enhance the case study:

- Screenshots of the website showing usability issues
- Persona boards
- Card sorting results
- User journey maps
- Accessibility testing screenshots
- Before/after comparisons

## How to Add Images

1. Take a screenshot of page 1 from your PDF (the cover with the Pedal Stoke homepage screenshot)
2. Crop it to focus on the main visual
3. Save it as `hero.jpg` in this directory
4. Optionally, extract other relevant visuals from your PDF and add them with descriptive names

Once you add the hero image, the case study card will display properly on your portfolio!
